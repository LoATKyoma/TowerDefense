# 一个简单的塔防游戏

## 开发背景

大二下学期软件综合实习题目，我与小组成员选择使用Unity平台开发一个塔防游戏

## 游戏内容

经典的塔防游戏方式，设计了四个关卡可供玩家选择，每个关卡玩家都要扛十波怪物的入侵。游戏内还藏了几首奇妙的音乐，假如不喜欢请勿打开XD，在游戏中你可以

* **构建你的防御系统**  
  >
  * 建造你喜欢的防御塔
  >
  * 升级防御塔，不同防御塔有不同的功能和伤害属性，请注意搭配才能消灭敌人
  >
  * 有时候，不得已你也可以销毁防御塔，不过不会有任何的返还哦
>
* **经济的来源——消灭敌人**  
  >
  * 防御塔可不是白给的，你需要金钱来完成建造
  >
  * 消灭敌人你才能获得金钱，否则你将在敌人的入侵下消亡
  >
  * 合理的花费金钱提升防御塔才是通关的诀窍
>
* **砸！哇路多**
  >
  * 操作不过来你可以停止，按空格或者ui上的暂停
  >
  * 镜头不够宽阔了？可以使用滚轮调节镜头的大小，同时可以用wasd移动镜头
  >
  * 打不过了可以用重开哦
